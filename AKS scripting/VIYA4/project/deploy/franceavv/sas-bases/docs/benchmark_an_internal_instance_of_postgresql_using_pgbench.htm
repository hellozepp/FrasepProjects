Benchmark an Internal Instance of PostgreSQL Using pgbench

Overview

If you have an internal instance of PostgreSQL, SAS Viya provides a tool, pgbench, to benchmark the PostgreSQL database. For information about how to use pgbench and the YAML file associated with it, see “sas-crunchy-data-pgbench” in SAS Viya Deployment Guide

Additional Resources

For more information about internal and external instances of PostgreSQL, see “Internal versus External PostgreSQL Instances” in SAS Viya Deployment Guide
